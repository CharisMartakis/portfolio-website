Project Overview

This project is a digital recreation of a Piet Mondrian painting using HTML and CSS. The design captures the essence of Mondrian's style, characterized by a grid of vertical and horizontal black lines and the use of primary colors (red, blue, and yellow) along with white and black.

Technologies Used

  • HTML5
  • CSS3

Features

  • Responsive design that adapts to different screen sizes.
  • Use of CSS Grid for layout management.
  • Clean and minimalistic aesthetic in line with Mondrian's artistic principles.
