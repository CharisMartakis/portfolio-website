Dice Game — Details

My first JavaScript project! This interactive dice game showcases fundamental JavaScript skills including DOM manipulation, random number generation, and dynamic content updates. Players refresh the page to roll two dice, and the game automatically determines and displays the winner. This project marks my entry into JavaScript development and demonstrates how to manipulate the DOM to create engaging, interactive web experiences.

  • Technologies: HTML5, CSS3, JavaScript
  • DOM manipulation using querySelector and setAttribute
  • Dynamic content updates and conditional logic
  • Random number generation with Math.random()
  • Interactive gameplay and user experience

This project demonstrates core JavaScript fundamentals and DOM manipulation skills essential for modern web development.
